# OIBSIP_python_taskno3

### Objective

To develop a Python program that calculates a person's Body Mass Index (BMI) using their weight and height, and classifies the BMI into standard health categories.

### Steps Performed

1. Defined a function to calculate BMI using weight and height.
2. Defined a function to classify the BMI value into health categories.
3. Accepted the user's weight (kg) and height (m) as input.
4. Validated that the entered values were positive numbers.
5. Calculated the BMI and determined the corresponding category.
6. Displayed the BMI value and its health classification.
7. Handled invalid (non-numeric) input using exception handling.

### Tools Used

* **Programming Language:** Python
* **Functions:** User-defined functions (`calculate_bmi()`, `classify_bmi()`)
* **Exception Handling:** `try-except` for input validation
* **Development Environment:** Any Python IDE (e.g., IDLE, VS Code, PyCharm, or Jupyter Notebook)

### Outcome

A Python-based BMI Calculator was successfully developed. It accurately computes the user's BMI, classifies it into standard health categories (Underweight, Normal weight, Overweight, or Obese), and provides input validation for reliable and user-friendly operation.
